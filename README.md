## Installation

#### Composer install
```sh
$ composer install
```

#### Laravel key generate
```sh
$ php artisan key:generate
```

#### Node module install
```sh
$ npm i
```

#### Mix build
```sh
$ npm run dev
```

#### Database migrate and seed
```sh
$ php artisan migrate --seed
```

#### Run server
```sh
$ php artisan serve
```
